Required adjustments before downloading E-Grades

In order to download E-Grades, a grading scheme must be enabled.

Set Grading Scheme

A grading scheme must be enabled to calculate student letter grades. You can choose to enable the default grading scheme, or manually enable a custom grading scheme.

Default Grading Scheme

Select to enable and use the default 'Letter Grades +/-' grading scheme below and then click on 'Continue' below.

Custom Grading Scheme

If you need to choose an alternative grading scheme for your course site other than the default scheme, please edit and enable the grading scheme in the page for this course site. Once this step is completed, return here and try again.

For more information see How do I enable a grading scheme for my course?

In order to download E-Grades, all assignments need to be unmuted.

Unmute Assignments
